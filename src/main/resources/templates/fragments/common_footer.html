도서 쇼핑몰 프로젝트

저희 byeol23.shop은 인기 웹툰 '*23#'에서 영감을 얻은 이름을 사용한 Spring 기반의 도서 쇼핑몰입니다. 도서 검색, 카테고리 탐색, 리뷰 작성, 그리고 안전한 주문 및 결제 프로세스를 모두 구현하여 풍부한 사용자 경험을 제공합니다. Java 스프링의 확장성을 바탕으로 제작된, 기능적으로 완전한 형태의 E-commerce 서비스입니다.
